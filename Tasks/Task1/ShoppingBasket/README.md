# README ShoppingBasket

This contains the ShoppingBasket class, along with a small test suite with
a demonstration test case.

## Javadocs

```java
$ gradle javadoc
```

## Building

```java
$ gradle build
```

## Testing

```java
$ gradle test
```

## Report Coverage

```java
$ gradle jacocoTestReport
```

## Show Tasks

```
$ gradle tasks
```
